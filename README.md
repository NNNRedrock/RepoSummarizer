# RepoSummarizer
## About:
RepoSummarizer is a tool that outputs a summary of a Github repo using commits, pull requests and issues.
## Repo Summary Output :
* Commits Summary
* Pull Requests Summary
* Issues Expertise Level 
* Pie Chart of Issues Expertise Level
